## Requirements

* Xcode 14.0+, iOS 12.0, tvOS 12.0, OSX 10.13, watchOS 3.0

## Installation

CXWebSocket is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'CXWebSocket'
```
